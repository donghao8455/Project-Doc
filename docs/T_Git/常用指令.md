## 常用指令

在`Git`命令行中，可以使用的命令类似`ubuntu`中的命令

|    命令    |                   描述                   |
| :--------: | :--------------------------------------: |
|    `cd`    |              进入某个文件夹              |
|   `pwd`    |              查看当前的目录              |
|  `mkdir`   |                创建文件夹                |
|  `touch`   |            创建一个空白的文件            |
|  `ls -a`   | 查看当前文件夹下的所有文件，包括隐藏文件 |
| `rm -rf *` |        删除当前文件夹下的所有文件        |

`Git`中内置了`vim`编辑器，可以使用`vim`来进行文件的编辑：

| 按下的键 |      描述      |
| :------: | :------------: |
|   `i`    |  启用插入模式  |
|  `Esc`   |  退出编辑模式  |
|   `wq`   | 保存并退出文件 |
