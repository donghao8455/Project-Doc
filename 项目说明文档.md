| 项目属性 | 说明 |
|---|---|

## 五、站点配置

### 5.1 导航栏

顶部导航栏包含 **6 个一级菜单**：回到首页、前端、数据库、后端、网络通信、开发工具、经验积累，每个菜单下设对应技术条目的下拉选项。

### 5.2 侧边栏

根据 URL 路径前缀自动匹配对应分类，展示该技术下的所有文章列表。当前共配置了 **48 个侧边栏分组**。

### 5.3 首页定制

- **Hero 区域**：展示站点名称 "DongHao's"、Slogan "记录学习，分享知识，个人在线项目文档库"、Logo 图片
- **快捷入口**：个人简介、笔记更新历程、Java面试、AI大模型开发
- **Features 卡片**：不断更新、开源共享、联系方式
- **HomeComponent 组件**：展示个人开源项目卡片（含项目描述、在线体验链接、GitHub/Gitee 仓库链接）

### 5.4 其他配置

- 搜索：本地搜索 (`provider: 'local'`)
- 社交链接：GitHub (`donghao8455`)、Gitee (`donghao8455`)
- 页脚：MIT License，Copyright 2021-present DongHao's

## 六、构建与部署

| 命令 | 用途 |
|---|---|
| `npm run dev` | 启动本地开发服务器 |
| `npm run build` | 构建静态站点文件 |
| `npm run preview` | 预览构建后的站点 |

构建配置中 `srcDir` 设为 `./docs/`，文档源文件位于 `docs` 目录下。
